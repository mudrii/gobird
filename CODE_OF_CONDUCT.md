# Code of Conduct

## Our standard

Participation in the `gobird` project should be respectful, direct, and constructive.

Examples of expected behavior:
- assume good intent
- focus on technical issues, not people
- give clear, actionable feedback
- accept correction without hostility

Examples of unacceptable behavior:
- harassment or intimidation
- personal attacks
- discriminatory language
- doxxing or publishing private information
- repeated bad-faith disruption

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributors that violate this code of conduct.

For conduct concerns, contact the maintainers through the security/contact path in [SECURITY.md](SECURITY.md).
